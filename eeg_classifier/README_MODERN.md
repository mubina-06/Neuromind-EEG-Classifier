The app is highly customizable:

- **Layout**: Adjust column ratios and spacing
- **Charts**: Customize Plotly themes and colors
- **Content**: Add your own metrics and visualizations

## 📞 Support

For questions or issues with the modern interface:
1. Check the console for error messages
2. Ensure all dependencies are installed
3. Verify model files are present in `models/` directory
4. Test with sample data from `data/spectrograms/`
