# Assets Directory

This directory contains visual assets for the NeuroMind EEG Classifier project.

## Structure

- **`demo_screenshots/`** - Screenshots of the web application interface
- **`architecture_diagrams/`** - System architecture and flow diagrams

## Adding Assets

When contributing screenshots or diagrams:

1. Use high-resolution images (minimum 1920x1080 for screenshots)
2. Save in PNG format for screenshots, SVG preferred for diagrams
3. Use descriptive filenames (e.g., `streamlit_main_interface.png`)
4. Compress large images to keep repository size manageable
5. Update README files when adding new visual assets

## Screenshot Guidelines

- Capture the full application interface
- Use consistent browser/window sizing
- Include realistic sample data in demonstrations
- Avoid including personal information in screenshots

## Diagram Guidelines

- Use consistent color schemes matching the brand
- Include clear labels and legends
- Maintain readability at different sizes
- Provide both light and dark theme versions when applicable
